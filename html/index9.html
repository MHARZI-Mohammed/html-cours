10. Validation HTML

Qu'est-ce que la validation ?

C'est le processus de vérification de votre code HTML par rapport aux **standards définis par le W3C**. Un code valide assure une meilleure compatibilité et accessibilité.

Erreurs courantes :

Oublier de fermer une balise (ex: <p>), ne pas inclure l'attribut alt dans une balise <img>, ou utiliser une balise non standard.

Pratique : Correction d'erreur

Le code ci-dessous contient une erreur de validation courante. Identifiez et corrigez-la :

<html>
<head>
    <title>Page</title>
</head>
<body>
    <p>Ceci est un paragraphe
</body>
</html>

Correction :

L'erreur est le paragraphe non fermé. On a ajouté </p>.

<html>
<head>
    <title>Page</title>
</head>
<body>
    <p>Ceci est un paragraphe</p>
</body>
</html>
