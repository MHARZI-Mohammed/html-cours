8. Balises sémantiques

Le sens avant le style :

Les balises sémantiques (comme <header>, <nav>, <main>, <footer>) décrivent la **nature** de leur contenu, aidant les moteurs de recherche et les outils d'accessibilité à comprendre la structure de la page.

Balises de contenu :

<article> est pour le contenu indépendant (comme un post de blog). <section> est un regroupement thématique général.

Pratique : Le bloc de navigation

Encadrez une liste de liens ("Accueil", "Produits", "Contact") avec la balise sémantique appropriée.

Correction :

<nav>
  <ul>
    <li><a href="#">Accueil</a></li>
    <li><a href="#">Produits</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
