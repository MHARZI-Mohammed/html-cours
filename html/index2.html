3. Balises de texte

Hiérarchie et importance :

Utilisez <h1> à <h6> pour les titres et <p> pour les paragraphes. Les balises <strong> (importance forte, souvent gras) et <em> (emphase, souvent italique) sont sémantiques.

Les sauts :

La balise <br> est utilisée pour un simple saut de ligne. La balise <hr> insère une ligne horizontale pour séparer le contenu thématiquement.

Pratique : Formatez un texte

Écrivez le code pour obtenir : un titre de niveau 2, suivi d'un paragraphe où le mot "attention" est en gras (strong) et le reste du paragraphe est sur deux lignes distinctes.

Correction :

<h2>Rapport quotidien</h2>
<p>
    Veuillez prêter <strong>attention</strong> au rapport.
    <br>
    Il contient toutes les données récentes.
</p>
