1. Introduction au HTML

Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le **langage de balisage** fondamental pour structurer le contenu web. Il n'est pas un langage de programmation, mais il définit la hiérarchie de vos éléments (titres, paragraphes, médias).

Exemple de base :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma première page</title>
</head>
<body>
    <h1>Bonjour le monde !</h1>
</body>
</html>

Pratique : Votre première balise !

Écrivez uniquement la balise HTML nécessaire pour afficher le texte "Bienvenue sur mon site" comme un **titre de niveau 1**.

Correction :

<h1>Bienvenue sur mon site</h1>
