9. Intégration multimédia

Audio et Vidéo :

Les balises <audio> et <video> permettent d'intégrer nativement les médias. L'attribut controls est essentiel pour donner le contrôle à l'utilisateur.

Iframes (Contenu externe) :

La balise <iframe> est utilisée pour intégrer du contenu d'une autre source (par exemple, une vidéo YouTube ou une carte Google Maps).

Pratique : Intégration Vidéo

Écrivez le code pour intégrer une vidéo nommée "demo.mp4" d'une largeur de 640 pixels, en vous assurant que l'utilisateur a les contrôles de lecture.

Correction :

<video width="640" controls>
  <source src="demo.mp4" type="video/mp4">
  Votre navigateur ne supporte pas la balise vidéo.
</video>
