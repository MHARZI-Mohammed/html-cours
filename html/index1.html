2. Structure de base d'un document HTML

Les balises fondamentales :

Le squelette de toute page web est défini par <!DOCTYPE html>, <html>, <head> (pour les métadonnées) et <body> (pour le contenu visible).

La balise <head> :

Elle contient des informations vitales pour le navigateur et les moteurs de recherche, comme le titre de la fenêtre (<title>) et l'encodage des caractères (<meta charset="UTF-8">).

Pratique : Complétez la structure

Écrivez la structure complète d'un document HTML5 qui affiche le titre "Mon Portfolio" dans la barre du navigateur.

Correction :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Portfolio</title>
</head>
<body>
    <!-- Contenu ici -->
</body>
</html>
